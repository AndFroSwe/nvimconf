# Neovim Config

Based of [Kickstart](https://github.com/nvim-lua/kickstart.nvim).

## Installation notes


Setup relies heavily on external tools. Make sure the following are installed.

```fzf ripgred fd tree-sitter npm cargo```

### Linux

On linux, some minor stuff also needs to be available. On Arch with Wayland, install ```wl-clipboard``` to get clipboard
integration.
